# PyTorch Learning Project

This repository contains basic PyTorch implementations of regression models.

## Contents

### 1. Linear Regression
- `linear_regression.py`: Implementation of linear regression from scratch.
- `linear_regression.md`: Detailed documentation of the implementation.

### 2. Softmax Regression (FashionMNIST)
- `fashion_mnist.py`: Implementation of a Softmax classifier for the FashionMNIST dataset.
- `fashion_mnist.md`: Detailed documentation and explanation of Softmax regression.

## Environment
- PyTorch
- torchvision
- d2l (Dive into Deep Learning)
- matplotlib
